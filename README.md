# hexo
## hexo 使用方法:
1. npm install hexo-cli -g
2. hexo init blog
3. cd blog
4. npm install
5. hexo server

## 项目初始化
1. git clone url: [https://github.com/boylufeng20141007/FEEntertainment.git](https://github.com/boylufeng20141007/FEEntertainment.git)
2. cd ~/FEEntertainment
3. npm install
3. hexo server
